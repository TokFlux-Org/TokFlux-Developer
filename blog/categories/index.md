# 分类

TokFlux 使用少量、明确的分类，让文章列表更容易扫描和维护。

| 分类 | 作用 | 从这里开始 |
| --- | --- | --- |
| 教程 | 构建与集成指南 | [通过 OpenAI 兼容网关处理流式输出](/blog/tutorials/streaming) |
| 错误排查 | 排查手册与问题定位 | [排查 401 无效供应商配置](/blog/errors/401-provider-config) |
| 对比分析 | 架构与产品取舍 | [网关与直连供应商接入的比较](/blog/comparisons/gateway-vs-direct) |
| 产品公告 | 发布与博客更新 | [TokFlux 创作者奖励申请指南](/blog/announcements/creator-reward) |

## 元数据示例

```yaml
categories:
  - 教程
tag:
  - OpenAI 兼容
  - 流式输出
```
