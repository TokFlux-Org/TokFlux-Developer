# 标签

标签为博客提供了分类之外的第二层信息检索能力。

## 核心标签

- `OpenAI 兼容`
- `网关`
- `流式输出`
- `客户端 SDK`
- `可观测性`
- `错误排查`
- `路由`
- `架构`
- `Cloudflare`
- `上线`

## 使用方式

- 分类保持稳定且相对宽泛。
- 标签聚焦具体技术点、关注点或运维层面。
- 文章可以同时携带产品、基础设施与工作流标签，而不破坏主导航结构。
