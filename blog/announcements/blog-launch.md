# TokFlux 技术博客上线

TokFlux 正在维护一个轻量技术博客，把教程、故障排查、架构判断与产品更新放在同一个清晰的阅读入口里。

## 为什么先做轻量博客

比起一开始就铺开完整门户，轻量博客更适合快速沉淀真实问题：怎么接入、哪里容易卡、什么方案值得复用。

## 第一版基础能力

- 基于 `@sugarat/theme` 的博客系统
- 统一导航、样式、搜索、RSS 与站点地图
- 面向开发者阅读习惯优化的深色模式、代码展示与文章分类
- 保留文档入口，但不让它挤占博客主线

## 下一步

接下来会优先补齐高频问题与集成教程。需要长期稳定的 API 内容仍然放在文档里，博客则继续承载更灵活的工程经验。
