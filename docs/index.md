# 文档概览

这里当前刻意保持轻量，只提供 TokFlux 文档的基础框架，方便内容编写与站点建设并行推进。

## 规划中的部分

- [指南](/docs/guide/) 用于接入说明与平台概念介绍
- [API](/docs/api/) 用于接口与 Schema 参考
- [示例](/docs/examples/) 用于 SDK 用法与请求模式示例
- [定价](/docs/pricing/) 用于套餐与计费说明
- [更新日志](/docs/changelog/) 用于版本演进记录

## 设计说明

文档侧只保留必要入口，不参与博客首页的信息层级。博客负责承载教程、排错和更新，文档负责长期稳定的参考内容。

## 部署特性

- 面向 Cloudflare Pages 的静态构建输出
- 与博客共享本地搜索和基础元信息
- 文档入口保持在主导航末尾
- 为内容作者提供干净的占位入口
